# Agent

AI Agent 智能体相关知识。

## 概念

**定义** 智能体是指**具有自主决策能力和自我学习能力的计算机程序或机器人**。智能体可以通过感知环境、分析信息、制定决策并执行行动来完成任务。

智能体通常包括感知模块、决策模块和执行模块。
