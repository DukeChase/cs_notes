## 属性选择器
| 选择器                  | 示例                              | 描述                                                                                                           |
| -------------------- | ------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| `[_attr_]`           | `a[title]`                      | 匹配带有一个名为_attr_的属性的元素——方括号里的值。                                                                                |
| `[_attr_=_value_]`   | `a[href="https://example.com"]` | 匹配带有一个名为_attr_的属性的元素，其值正为_value_——引号中的字符串。                                                                   |
| `[_attr_~=_value_]`  | `p[class~="special"]`           | 匹配带有一个名为_attr_的属性的元素，其值正为_value_，或者匹配带有一个_attr_属性的元素，其值有一个或者更多，至少有一个和_value_匹配。<br><br>注意，在一列中的好几个值，是用空格隔开的。 |
| `[_attr_\|=_value_]` | `div[lang\|="zh"]`              | 匹配带有一个名为_attr_的属性的元素，其值可正为_value_，或者开始为_value_，后面紧随着一个连字符。                                                   |
