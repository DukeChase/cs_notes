# 概率分布
当然可以！我们用一个生活中的例子来理解“概率分布”。

想象你有一个公平的六面骰子（就是平时玩桌游那种），每一面分别是1到6点。当你掷一次骰子，可能出现的结果有：1、2、3、4、5、6。每个结果出现的“可能性”是一样的，都是1/6。

**概率分布**，说白了，就是把所有可能的结果和它们各自发生的“可能性”（也就是概率）列出来的一种方式。

在这个骰子的例子中，概率分布就是：

- 出现1的概率是 1/6
- 出现2的概率是 1/6
- ……
- 出现6的概率是 1/6

这个列表或者这张“地图”，就叫**概率分布**。它告诉我们：某个事情可能发生哪些结果，以及每个结果有多大概率发生。

再举个不一样的例子：比如你每天早上遇到红灯的概率是70%，绿灯是30%。那“等红绿灯”这件事的概率分布就是：

- 红灯：70%
- 绿灯：30%

所以，**概率分布 = 所有可能结果 + 每个结果对应的概率**。

根据结果是“离散的”（比如骰子点数）还是“连续的”（比如人的身高、气温），概率分布也分不同类型，比如：

- **离散型分布**：像骰子、抛硬币——结果是一个个分开的。
- **连续型分布**：像身高、考试分数——结果可以在一个范围内任意取值，这时候用的是“概率密度”而不是直接的概率。

但不管哪种，核心思想都一样：**描述一件事各种结果的可能性大小**。

希望这个解释让你觉得概率分布没那么抽象啦！ 😊
